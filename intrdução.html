Pre-requisitos: Ter um conhecimento minimo em lógica de programação

-Variavel: Em Python, uma variável é um local na memória onde você pode armazenar valores. Por exemplo, você pode criar uma variável chamada 'idade' e atribuir a ela o valor 20. Ex: idade = 20

-String: Uma string é um tipo de dado que é usado para representar texto. Em Python, você pode criar uma string colocando o texto entre aspas simples ou duplas. Ex: nome = "João"


Tipos de dados: Em Python, existem vários tipos de dados, incluindo inteiro (int), flutuante (float), string (str), booleano (bool), lista (list), tupla (tuple), conjunto (set) e dicionário (dict). Cada tipo de dado tem suas próprias características e usos. Por exemplo, os inteiros são usados para representar números inteiros, as strings são usadas para representar texto, e as listas são usadas para armazenar várias peças de dados em uma única variável.

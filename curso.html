Introdução de Python

Tópicos

  • Variáveis
  • Comando print
  • Condicionais (If, elif, else)
  • Loops (while e for)
  • Estrutura de dados (lista)

Pre-requisitos: Vontade de aprender e um PC.

-Variavel: Em Python, uma variável é um local na memória onde você pode armazenar valores. Por exemplo, você pode criar uma variável chamada 'idade' e atribuir a ela o valor 20. Ex: idade = 20

-String: Uma string é um tipo de dado que é usado para representar texto. Em Python, você pode criar uma string colocando o texto entre aspas simples ou duplas. Ex: nome = "João"

Tupla, conjunto e dicionário não serão abordados na explicação

Tipos de dados: Em Python, existem vários tipos de dados, incluindo inteiro (int), flutuante (float), string (str), booleano (bool), lista (list), tupla (tuple), conjunto (set) e dicionário (dict). Cada tipo de dado tem suas próprias características e usos. Por exemplo, os inteiros são usados para representar números inteiros, as strings são usadas para representar texto, e as listas são usadas para armazenar várias peças de dados em uma única variável. Print: O comando print é usado para imprimir valores na tela. Ex: print("Olá, mundo!")

Lista: Uma lista é uma coleção de itens que podem ser de diferentes tipos. As listas são mutáveis, o que significa que você pode adicionar, remover e alterar itens depois que a lista é criada. Ex: frutas = ["maçã", "banana", "laranja"]

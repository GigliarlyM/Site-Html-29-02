Vamos ensinar como fazer um commit com o git Bash.

Já tendo o aplicativo do Git Bash em seu aparelho

Os passos são esses:

  1. Caminhe até a pasta do projeto

  2. Posteriormente digite git init

  3. Apos digite git add . (Para que seja adicionado todos os arquivos no commit)

  4. Por fim digite git commit -m "o nome de seu commit"

  5. Agora é só fazer o push git push origin HEAD:main ("main" é o nome de sua branch)

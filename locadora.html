Introdução da locadora

Agora você vai aprender como estruturamos nossa locadora:

Começamos definindo uma variável opção valor 1, ela será necessária para o funcionamento While.

Criamos também uma lista chamada listaFilme é nela que nossos filmes serão armazenados.

Aqui está o While, enquanto a variável opção for diferente de 5, nosso programa continuará execuntando, contudo se digitado número 5,saímos do programa.

Printamos nosso menu (\n serve para pular a linha).

Variável opção recebe um input(entrada), o usuário precisa informar algum valor numérico.


Funcionalidades

Vamos falar também sobre as funcionalidades do nosso programa:

Opção 1

Opção 1 lista todos os filmes encontrados na lista

Criamos um for e ele é responsável por trazer todos os itens colocados na listaFilmes, logo ápos, printamos os filmes.


Opção2

Adiciona um novo filme na lista

A variável novofilme recebe um filme digitado pelo usuário.

A funcionalidade .append(), adiciona o elemento na lista.

Printamos uma mensagem.


Opção 3

Remove um filme da lista

A variável removerFilme recebe o nome de um filme digitado pelo usuário.

É verificado se existe o filme na lista, para então removê-lo e printar uma mensagem.

Se não existir o filme digitado, uma mensagem de aviso é exibida.


Opção 4

Pesquisa um filme na lista

A variável pesquisarFilmes recebe o nome de um filme digitado pelo ususario.

É verificado se o filme pesquisado consta na lista, para exibí-lo.

Se o filme pesquisado não for encontrado, uma mensagem de aviso é exibida.


Opção 5

O programa é finalizado

Ao digitar o número 5, o While é interrompido e o programa é finalizado com uma mnensagem.


Exceção

Se um número digitado pelo usuario não estiver dentre as opções fornecidas pelo programa uma
mensagem de aviso é exibida.

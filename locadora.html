Introdução da locadora

Agora você vai aprender como estruturamos nossa locadora:


opcao = 1

listaFilmes = []

while (opcao != 5):

print("\n1)Listar filmes \n2)Novo filme \n3)Remover \n4)Pesquisar \n5)Sair\n")

opcao = int(input("Digite uma opção: "))

Começamos definindo uma variável opção valor 1, ela será necessária para o funcionamento While.

Criamos também uma lista chamada listaFilmes é nela que nossos filmes serão armazenados.

Observe o While, enquanto a variável opção for diferente de 5, nosso programa continuará executando, contudo, se digitado o número 5, saímos do programa.

Variável opção recebe um input(entrada), o usuário precisa informar algum valor numérico.

Funcionalidades

Vamos falar também sobre as funcionalidades do nosso programa:

Opção 1

lista todos os filmes encontrados na lista

Criamos um for e ele é responsável por trazer todos os itens colocados na listaFilmes, logo ápos, printamos os filmes.

Opção2

Adiciona um novo filme na lista

A variável novofilme recebe um filme digitado pelo usuário. A funcionalidade .append(), adiciona o elemento na lista.

Opção 3

Remove um filme da lista

A variável removerFilme recebe o nome de um filme digitado pelo usuário, para então removê-lo e printar uma mensagem.

Opção 4

Pesquisa um filme na lista

A variável pesquisarFilmes recebe o nome de um filme digitado pelo usuário. É verificado se o filme pesquisado consta na lista, para exibí-lo.

Opção 5

O programa é finalizado

Ao digitar o número 5, o while é interrompido e o programa é finalizado com uma mensagem.

Exceção

Se um número digitado pelo usuario não estiver dentre as opções fornecidas pelo programa uma mensagem de aviso é exibida.
